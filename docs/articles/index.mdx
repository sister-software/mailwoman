# Mailwoman

Mailwoman parses postal addresses into structured components and geocodes them against open
gazetteer data. It runs locally, in Node.js or the browser, with no API key and no address
leaving your process.

## I want to parse addresses

Install it, parse your first address, and read the output. About five minutes.

**[Get started →](./getting-started.mdx)**

## I want to understand how this works

Follow one address through the whole parse — subword pieces, per-label scores, gazetteer
priors, decoding, and the component tree the resolver receives.
