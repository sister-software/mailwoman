## The verdict took three batteries — the record, in order

1. **Battery 1: FAIL po_box 60.4/70.** Row audit: dotted leaders (P.O. / C.P. / B.P.) missed at
   **98%, all truncations at the first period**; plain leaders passed 84%. Root cause is
   structural: the corpus alignment tokenizer drops standalone punctuation, so NO training row
   can label the dots inside "P.O. Box" — the model labels every letter piece `po_box` at 0.93+
   confidence and the span shatters at decode. Ten-times exposure had moved the number +2.9; it
   was never a dose problem. (This battery was also killed mid-int8-leg by a mid-run edit of the
   runner script — operator note: scripts with live instances are now immutable; the fp32 record
   survived and carries the FAIL evidence.)
2. **Corrective 1 — the span bridge** (`neural/span-bridge.ts`): merge same-tag fragments across
   short punctuation gaps, absorbing the O-labeled punctuation pieces; space-only gaps excluded
   (the Saint-Albans guard). Same weights: po_box 60.4 → 87.0. **Battery 2: FAIL fr.postcode
   99.0/99.5** — the bridge merged `"47110, 9016"`-style postcode + house-number fragments
   across commas on six FR golden rows (the model double-labels the following number; the comma
   was the only thing keeping the spans honest).
3. **Corrective 2 — separator exclusion**: bridgeable gaps are intra-token punctuation only
   (period/hyphen/slash/apostrophe). FR postcode restored to its two pre-existing CA-alnum
   misses; po_box improved further (87.0 → 89.1 — comma false-merges removed there too).
   **Battery 3: PASS 17/17.**

Both correctives are decode-side, deterministic, regression-tested (8 bridge tests), and cost
zero GPU. The night's total GPU spend for the verdict: $0 beyond the operator-GO'd training run.

## What this release means for the parity campaign

po_box, cedex, and intersections were the scorecard's last empty rows. They are no longer empty:
89.1 / 96.1 / 100 on real-OOD evals, at precision ≥ 95 everywhere. The v4.3.0 arena debt (#513)
is closed at gate level. The remaining v0-vs-neural gaps are quality gradients and edge formats,
not missing capabilities — the post-parity agenda (#478 arbitration, decomposition,
geocoder table stakes) starts from here.

Follow-ups filed/banked: the conventions loss-mask rides the next full run with a pre-registered
FR-region floor; the dotted-surface class should eventually be fixed at the corpus format level
(char-offset labels — the bridge is containment, not cure); the double-postcode emission the
comma incident exposed is a training-distribution question for the next corpus pass.
