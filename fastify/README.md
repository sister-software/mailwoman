# @mailwoman/fastify

Fastify plugin for [mailwoman](https://mailwoman.sister.software) — install this package to
get geocoding, address parsing, and POI search routes in your Fastify app, backed by the
local mailwoman pipeline (no external geocoding service).

> **v0.0.1 is a name-reservation placeholder** published to configure npm Trusted
> Publishing. The first functional release ships with the next mailwoman minor.
